# Unsplash Viewer
## 개요
Unsplash에서 이미지 리스트와 이미지를 가져와 표시해주는 간단한 뷰어입니다.
평소에 외부 API을 이용한 개발을 잘 할일이 없어서 API 받아와서 표시하는 부분까지 외부 라이브러리 없이 쓰는 것을 목표로 함.
